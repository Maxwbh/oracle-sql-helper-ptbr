⬇ Instalação via Claude.ai

1
Baixe os arquivos .skill da página de releases — ou clique nos links de download acima.
2
No Claude.ai → Settings → Skills → arraste o arquivo ou clique em Upload Skill.
3
Repita para cada skill desejada. Instale todas as 7 para cobertura completa da stack Oracle.
4
Teste — diga ao Claude: "Crie um package Oracle para processar pagamentos seguindo Trivadis"
